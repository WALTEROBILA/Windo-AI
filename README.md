# Windo

An AI-powered resume and cover letter optimization platform that tailors application documents to specific job advertisements.

## Features

- Paste a job advertisement URL
- Upload a resume and cover letter
- Automatically extracts the job description
- Generates an ATS-optimized resume
- Generates a personalized cover letter
- Preserves factual information from the original documents
- Produces downloadable Microsoft Word documents

## Tech Stack

- Python
- Streamlit
- LangGraph
- LangChain
- Groq LLM
- Selenium
- BeautifulSoup
- python-docx
- PyPDF
